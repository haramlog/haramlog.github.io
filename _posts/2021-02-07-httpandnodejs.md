## 1. URL

```
http://opentutorials:3000/main?id=HTML&page=12
```

 * http / 프로토콜
 * opentutorials / 호스트(도메인)
 * 3000 / 포트번호
 * main / PATH(어떤 디렉토리 안에 어떤 파일)
 * id=HTML&page=12 / 쿼리스트링
 
 id 값이 무엇이냐에 따라서 사용자에게 적당한 값을 보여주어야 한다.
